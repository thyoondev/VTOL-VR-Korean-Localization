# 적용 방법

기본 설치 경로 : `C:\Program Files (x86)\Steam\steamapps\common\VTOL VR`

1. 게임 설치 경로로 이동합니다.
2. `Localization` 폴더를 열어주세요.
3. `ko`폴더를 열어주세요.
4. 다운로드 받은 압축파일을 풀고 `ko`폴더 안에 덮어씌워주세요.
